# КемьФест — сайт-помощник реализации проекта

Статический сайт для сопровождения проекта «Исторический фестиваль \"КемьФест\"».
Он помогает вести реализацию проекта: контрольные точки, статусы мероприятий, фактические показатели, ссылки, подтверждения, бюджет, документы и подготовку отчётности.

## Возможности
- рабочий стол с ключевыми показателями проекта;
- 12 мероприятий календарного плана;
- статусы «Не начато / В работе / Выполнено»;
- фактические даты, результаты, ссылки и заметки;
- контрольные точки и подсказки по ближайшим рискам;
- контроль участников, публикаций и просмотров;
- контроль исполнения бюджета 891 000 ₽;
- чек-лист отчётных документов;
- экспорт и импорт прогресса в JSON;
- печать или сохранение в PDF.

## GitHub Pages
Включите публикацию в `Settings → Pages`: `Deploy from a branch`, ветка `main`, папка `/root`.

После включения сайт будет доступен по адресу:
`https://mrkustov927-star.github.io/kemfest2026/`.
